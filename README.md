# Perc - Technical Test

PRUEBA ANGULAR

Requisitos:
1. Aplicar ruta “/books”, la cual nos ayudará a visualizar la lista de libros
2. Tabla de Visualización:
○ Implementa una tabla que muestre una lista de elementos.
○ Cada elemento debe tener un ID y un Nombre.
3. Filtrado:
○ Añade una barra de búsqueda que permita filtrar los elementos de la tabla por
nombre.
4. Formulario CRUD:
○ Implementa un formulario para crear nuevos elementos. El formulario debe tener
campos para el ID y el Nombre.
○ Permite editar un elemento existente al seleccionar una fila de la tabla y rellenar
el formulario con los datos de ese elemento.
○ Implementa la funcionalidad para eliminar un elemento de la lista al hacer clic en
un botón de eliminar en cada fila de la tabla.

5. Manejo de Datos en Memoria:
○ Los datos deben ser manejados en memoria, sin necesidad de una base de
datos o almacenamiento persistente.

Entregables:
● Asegúrate de que tu código sea limpio y esté bien comentado.
Criterios de Evaluación:
● Correcta implementación de las funcionalidades solicitadas.
● Uso adecuado de Angular 18 o superior
● Integración y uso efectivo de Tailwind CSS y Angular Material.
● Calidad y legibilidad del código.
